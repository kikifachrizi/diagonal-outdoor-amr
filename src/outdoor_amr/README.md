# Pengembangan Robot Outdoor (MAMR — Manufacture Autonomous Mobile Robot)

## Hardware Feature & Component

### Feature
1. Multi-sensor navigasi
2. Outdoor-indoor navigasi (outdoor first)

### Component (potential komponen yang akan dipakai)
1. LiDAR ✅
2. IMU ✅
3. GPS ✅
4. Multi-LiDAR (multi scanning)
5. Depth camera (Astra Pro / Kinect / ZED2i / RealSense)
6. Double webcam
7. Odometry

## Methodology
1. VO / VIO
2. 3D-SLAM
3. Navigating using GPS localization
4. Fusion sensor
   - IMU
   - LiDAR
   - Odometry
   - Camera
   - GPS
5. ORB-SLAM
6. Auto-mapping
7. Charging Docking
